PROJECTS
NODEJS GAME SERVER & CLIENT

Source: https://github.com/Sehrentos/nodejs-game-server

This is a MMO-like game server written in NodeJS. It uses MariaDB as a database and uses WebSockets for communication between clients and server.

This was a good learning experience for me, and it was fun to make.

We had so much fun drawing images together for the game and playing it with the kids.

BARCODE SCANNER FOR ANDROID

Source: https://github.com/Sehrentos/BarcodeReader

This project was made to scan barcodes from Android devices. It uses the Goolgle ML Kit library for barcode scanning and stores the results in a SQLite database.

I made this project because I wanted to learn how to use Google ML Kit in Android.
